PaiwanTalk Chrome/Edge 擴充程式範本\n\n使用方式（開發階段）：\n1. 確保後端已在本機啟動：\n   cd backend\n   uvicorn main:app --host 0.0.0.0 --port 8000 --reload\n\n2. 打開 Chrome 或 Edge：\n   - 進入 擴充功能 / 擴充套件 管理頁\n   - 開啟「開發人員模式」\n   - 選擇「載入未封裝項目」或「載入未封裝擴充功能」\n   - 指定本資料夾：extension/\n\n3. 使用方式：\n   - 右鍵選取網頁上的排灣語文字，選「使用 PaiwanTalk 翻譯選取文字」→ 會跳出 alert 顯示翻譯結果。\n   - 或點擊瀏覽器工具列中的擴充程式圖示，打開小視窗，在輸入框中貼上排灣語句子，按「翻譯成中文」。\n\n4. 若日後後端部署到正式網域，請修改：\n   - manifest.json 裡的 host_permissions\n   - background.js / popup.js 裡的 API_BASE URL\n
